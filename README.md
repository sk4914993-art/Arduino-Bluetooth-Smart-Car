🚗 Arduino Bluetooth Smart Car

📌 Overview

This project is a Bluetooth controlled smart car developed using Arduino UNO and L298N motor driver. The car can be controlled wirelessly through a mobile phone using Bluetooth commands. It is designed to move in different directions such as forward, backward, left, right, and stop.

This project demonstrates the basic concepts of embedded systems, motor control, wireless communication, and robotics.

---

⚙️ Components Used

- Arduino UNO
- L298N Motor Driver
- Bluetooth Module (HC-05 / HC-06)
- 2650mAh Li-ion Battery ×3
- 9V Battery
- 12V DC Motors ×4
- Wheels ×4
- Jumper Wires
- Battery Holder
- Chassis / Setup Board

---

🔧 Working Principle

The Bluetooth module receives commands from a mobile device and sends them to the Arduino UNO. Arduino processes these commands and controls the motor driver accordingly. The motor driver operates the DC motors to move the car in the desired direction.
